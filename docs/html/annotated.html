Data Structures

Here are the data structures with brief descriptions:
Rpe_AttributesSpecifies attributes for an RPE component instance during a Create call
Rpe_ClientClassConfigDefines structure for the client side configuration of a particular RPE implementation class
Rpe_ClientConfigDefines structure for the client side configuration of a particular RPE implementation
Rpe_ClientObjDefines structure for the client object. It is allocated in Rpe_create call
Rpe_ServerClassConfigDefines structure for the server side configuration of a particular RPE server class
Rpe_ServerConfigDefines structure for the server side configuration of a particular RPE server
Rpe_ServerObjDefines structure for an RPE instance server object
Rpe_ServerTaskConfigStatic configuration of tasks
XDM_FxnsDefines generic function pointer table for TI XDM interfaces. All XDM interfaces like IAUDDEC1, IAUDENC1, ISPHDEC1, ISPHENC1 etc. has function pointer tables IAUDDEC1_Fxns, IAUDENC1_Fxns, ISPHDEC1_Fxns, ISPHENC1_Fxns etc. that have three common entries: 1) ialg, 2) process, 3) control
XdmServer_ServerConfigDefines structure for the server side configuration of XDM server. It extends the base server configuration structure RPE_ServerConfig
XdmServer_ServerObjDefines structure for an XDM server object. It is used by server. It extends the base server object structure Rpe_ServerObj
