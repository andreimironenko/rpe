Enumeration Type Documentation

enum Rpe_BufferCpuAccessMode

Specifies how application processor accesses an input/output buffer.

Based on the access mode, middleware performs cache coherency operation.

Enumerator:
RPE_CPU_ACCESS_MODE_UNDEFINED 

Cpu access mode is undefined.

RPE_CPU_ACCESS_MODE_READ 

Accessed by processor for read. So, cache invalidate operation is performed before sending a buffer to a remote processor.

RPE_CPU_ACCESS_MODE_WRITE 

Accessed by processor for write. So, cache writeback-invalidate operation is performed before sending a buffer to a remote processor.

RPE_CPU_ACCESS_MODE_NONE 

Not accessed by processor. So, no cache operation is performed.

enum Rpe_CallDescType

Specifies the type of Remote Call Descriptor buffer.

Enumerator:

Control Call Descriptor buffer. Used for making Create, Control, Delete call

RPE_CALL_DESC_PROCESS 

Process Call Descriptor buffer
