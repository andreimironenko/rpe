Introduction

RPE 1.00.00.12 is a EA release of RPE 1.00.00.

RPE is a lightweight remote processor execute engine. It allows linux programs to call into algorithms running on the DSP. More information is present in the docs/ directory.

    Versioning

    All releases have 4 digits (M.mm.pp.bb). This includes GA and pre-releases (engineering, alpha/EA, beta, etc.).


    This product's version follows a version format, M.mm.pp.bb, where M is a single digit Major number, mm is 2 digit minor number, pp is a 2 digit patch number, and b is an unrestricted set of digits used as an incrementing build counter.

    To support multiple side-by-side installations of the product, the product version is encoded in the top level directory (e.g. rpe_1_00_00_00).

    Subsequent releases of patch upgrades will be identified by the patch number, ex. RPE 1.00.01.01 with directory rpe-1_00_01_01. Typically, these patches only include critical bug fixes.

    Please note that version numbers and compatibility keys are NOT the same. For an explanation of compatibility keys, please refer to the 'Upgrade and Compatibility Information' section.
